GRID-ALIGNED SPLATMAP SYSTEM FOR ROAD TEXTURE PAINTING
========================================================

This system allows players to place 1x1 road textures that blend smoothly with surrounding terrain
while following terrain contours perfectly.

SYSTEM OVERVIEW
---------------
- Aligned to World Grid: Each world unit (1x1) maps to specific pixel in splatmap
- Automatic Soft Falloff: Road tiles automatically create soft edges
- Terrain-Following: Roads painted into terrain shader follow contours perfectly
- Fast & Performant: Simple texture lookups, no extra geometry

IMPLEMENTATION DETAILS
----------------------

1. SPLATMAP RESOLUTION
   - 64x64 texture per 50x50 chunk (slight oversampling for smooth edges)
   - Each pixel covers ~0.78 world units
   - Memory: 64x64 RGBA = 16KB per chunk

2. PLACEMENT LOGIC
------------------

// When player places road at world position (x, z)
function placeRoadTile(worldX, worldZ) {
    // Find which chunk this belongs to
    const chunkX = Math.floor(worldX / 50);
    const chunkZ = Math.floor(worldZ / 50);

    // Convert to splatmap UV (0-1 range within chunk)
    const localX = (worldX % 50) / 50;
    const localZ = (worldZ % 50) / 50;

    // Convert to texture pixels
    const pixelX = Math.floor(localX * 64);
    const pixelZ = Math.floor(localZ * 64);

    // Paint with automatic falloff
    paintWithFalloff(chunk.splatmap, pixelX, pixelZ, {
        radius: 2,        // 2-pixel radius for soft edges
        strength: 1.0,    // Full strength at center
        falloff: 0.5      // How quickly it fades
    });
}

function paintWithFalloff(splatmap, cx, cz, options) {
    const { radius, strength, falloff } = options;

    for (let z = -radius; z <= radius; z++) {
        for (let x = -radius; x <= radius; x++) {
            const dist = Math.sqrt(x*x + z*z);
            if (dist <= radius) {
                const px = cx + x;
                const pz = cz + z;

                // Falloff calculation - smooth gradient
                const factor = Math.max(0, 1 - (dist / radius) * falloff);
                const value = strength * factor * factor; // Squared for smoother falloff

                // Set pixel (clamped to texture bounds)
                if (px >= 0 && px < 64 && pz >= 0 && pz < 64) {
                    const idx = (pz * 64 + px) * 4;
                    splatmap.data[idx] = Math.min(255, splatmap.data[idx] + value * 255);
                }
            }
        }
    }

    // Update the texture
    splatmap.needsUpdate = true;
}

// In shader - add variation to road texture
vec3 sampleRoadWithVariation(vec2 worldPos) {
    vec3 road1 = texture2D(uRoadTexture, worldPos * 0.5).rgb;
    vec3 road2 = texture2D(uRoadTexture, worldPos * 0.3 + vec2(0.5)).rgb;

    // Use world position for variation
    float blend = sin(worldPos.x * 0.5) * cos(worldPos.y * 0.5) * 0.5 + 0.5;
    return mix(road1, road2, blend * 0.3);
}

// Connect two points with road tiles
function createPath(start, end) {
    const steps = Math.max(
        Math.abs(end.x - start.x),
        Math.abs(end.z - start.z)
    );

    for (let i = 0; i <= steps; i++) {
        const t = i / steps;
        const x = Math.round(lerp(start.x, end.x, t));
        const z = Math.round(lerp(start.z, end.z, t));
        placeRoadTile(x, z);
    }
}

EROSION EFFECT (makes roads look worn):
// Add noise to splatmap for organic edges
function addErosion(splatmap, strength = 0.1) {
    for (let i = 0; i < splatmap.data.length; i += 4) {
        const noise = Math.random() * strength;
        splatmap.data[i] *= (1 - noise);
    }
}

7. CHUNK SPLATMAP MANAGEMENT
-----------------------------

class ChunkSplatmap {
    constructor(chunkX, chunkZ) {
        this.chunkX = chunkX;
        this.chunkZ = chunkZ;
        this.size = 64;

        // Create canvas for CPU-side manipulation
        this.canvas = document.createElement('canvas');
        this.canvas.width = this.size;
        this.canvas.height = this.size;
        this.ctx = this.canvas.getContext('2d');

        // Initialize to transparent (no roads)
        this.ctx.clearRect(0, 0, this.size, this.size);

        // Create THREE.js texture
        this.texture = new THREE.CanvasTexture(this.canvas);
        this.texture.minFilter = THREE.LinearFilter;
        this.texture.magFilter = THREE.LinearFilter;
        this.texture.wrapS = THREE.ClampToEdgeWrapping;
        this.texture.wrapT = THREE.ClampToEdgeWrapping;
    }

    paintTile(worldX, worldZ, options = {}) {
        // Convert world to local chunk coordinates
        const localX = worldX - (this.chunkX * 50);
        const localZ = worldZ - (this.chunkZ * 50);

        // Convert to texture pixels
        const px = (localX / 50) * this.size;
        const pz = (localZ / 50) * this.size;

        // Paint with gradient
        const gradient = this.ctx.createRadialGradient(px, pz, 0, px, pz, options.radius || 2);
        gradient.addColorStop(0, `rgba(255, 0, 0, ${options.strength || 1})`);
        gradient.addColorStop(1, 'rgba(255, 0, 0, 0)');

        this.ctx.fillStyle = gradient;
        this.ctx.fillRect(px - 3, pz - 3, 6, 6);

        // Update texture
        this.texture.needsUpdate = true;
    }

    save() {
        // Return base64 for saving to disk/server
        return this.canvas.toDataURL();
    }

    load(dataURL) {
        // Load from saved data
        const img = new Image();
        img.onload = () => {
            this.ctx.drawImage(img, 0, 0);
            this.texture.needsUpdate = true;
        };
        img.src = dataURL;
    }
}

HANDLING NON-TILEABLE TEXTURES
-------------------------------

Since the stone road texture doesn't tile well:

1. Texture Bombing (already in terrain.js):
   - Sample texture 3-4 times at different scales/rotations
   - Blend with noise-based weights

2. Detail Masking:
   - Use a tileable noise texture to add variation
   - Multiply/overlay on road texture

3. Edge Breakup:
   - Use noise in the splatmap painting
   - Creates organic, non-repetitive edges

4. Texture Atlas Variation:
   - Include 3-4 variations of road in atlas
   - Randomly select variation per tile
   - Blend at edges for seamless transition
